## Description

**MyProject** is a full-stack web application built using Node.js, Express, MongoDB, and EJS. It demonstrates core backend concepts such as routing, database integration, and server-side rendering, along with a structured MVC architecture.

This project was developed to strengthen full-stack development skills and showcase end-to-end application deployment.

---

## Features

- Server-side rendering with EJS
- RESTful routes and MVC structure
- CRUD operations using MongoDB
- Express.js backend
- Deployed on Render

---

## Tech Stack

- **Frontend:** EJS, HTML5, CSS3, JavaScript
- **Backend:** Node.js, Express.js
- **Database:** MongoDB, Mongoose
- **Deployment:** Render

---

## Run Locally

```bash
git clone https://github.com/kalyaniii04/myproject.git
cd myproject
npm install
npm start
```

App runs at: `http://localhost:3000`
